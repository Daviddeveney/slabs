# Macro Slab: M01 Shape The Model

## Status

`Done`

## Parent Brief

- [`../work-brief.md`](../work-brief.md)

## Objective

Define the first coherent version of the Slabs model so the framework has a shared vocabulary and a practical operating shape.

## Why It Matters

Without a clear model, future templates, automations, and agent behaviors will drift.

## In Scope

- core concept definitions
- relationship between briefs, macro slabs, micro slabs, tracking, and memory
- quality bar for slab sizing
- first canonical skill that embodies a default work-brief workflow
- canonical local project directory structure for multi-project use
- workspace-wide global context for references shared across projects

## Out of Scope

- production-grade schemas
- app implementation
- broad workflow-pack library design

## Dependencies

- none
