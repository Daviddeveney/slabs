# Context Note: Slabs Foundation

## Summary

Slabs Foundation is the canonical example project for the Slabs framework
itself. It captures the first public Markdown-native operating model, project
workspace conventions, templates, skills, plugins, and local review tooling.

## Why It Matters

This project is both product work and reference material. Future Slabs projects
copy its folder shape, tracking habits, durable memory conventions, and
suggestion-ready automation patterns.

## Source Or Origin

Created from the original `projects/slabs-foundation/work-brief.md` scope and
the active tracking hub.

## Current Relevance

Use this note when a project or automation needs a compact overview of why the
foundation project exists without reopening the legacy local brief.
